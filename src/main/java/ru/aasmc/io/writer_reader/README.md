# Writers and Readers

### Overview

- Writer (abstract)
  - BufferedWriter (default buffer size = 8192 bytes)
  - CharArrayWriter
  - FilterWriter (abstract)
  - OutputStreamWriter
    - FileWriter
  - PipedWriter
  - PrintWriter
  - StringWriter

- Reader (abstract)
  - BufferedReader (default buffer size = 8192 bytes)
    - LineNumberReader
  - CharArrayReader
  - FilterReader (abstract)
    - PushBackReader
  - InputStreamReader
  - PipedReader
  - StringReader
